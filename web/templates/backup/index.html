Backup

Run backups of configured sources to backup destinations.

Fields

  • Source — The backup source to run. Each source defines which folders and databases to back up.
  • Destination — Where to send the backup. Choose a specific destination or 'Default (all)' to back up to every configured destination.

Tips

  • Backups run in the background. You can monitor progress on the Running Tasks screen.
  • Each backup creates a timestamped snapshot on the destination, so you can restore any point in time.
  • Use retention cleanup to remove old snapshots.
