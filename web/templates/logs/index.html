Logs

View completed job history from the job registry.

Table columns

  • ID — Unique job identifier.
  • Label — Description of the job (e.g. target name).
  • Status — Outcome: Success, Failed, Skipped, or Unknown.
  • Started / Ended — When the job ran.
  • Duration — How long the job took.

Status meanings

  • Success — Job completed with exit code 0.
  • Failed — Job exited with a non-zero code.
  • Skipped — All targets were disabled.
  • Unknown — Job finished but exit code could not be determined.

Tips

  • The 20 most recent completed jobs are shown per page.
  • Running and queued jobs appear on the Running Tasks page.
  • Clear Logs removes all finished jobs and their log files.
