Destination Editor

Add or edit a backup destination.

Types

  • SSH — Remote server via SSH/rsync.
  • Local — Local directory or mounted drive.
  • S3 — Amazon S3 or compatible storage.
  • Google Drive — Google Drive via service account.

SSH fields

Host, port, user, and either SSH key or password authentication.

Common fields

  • Base path — Root directory for backups on the destination.
  • Bandwidth limit — Throttle transfer speed in KB/s (0 = unlimited).

Tips

  • For SSH, ensure the destination user has write access to the base path.
  • Use key-based auth for unattended backups.
