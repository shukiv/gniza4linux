Restore

Restore files and databases from a destination snapshot.

Fields

  • Source — The backup source to restore.
  • Destination — The destination that holds the snapshot.
  • Snapshot — The specific point-in-time snapshot to restore from. Loaded after selecting source and destination.
  • Restore location — Choose 'In-place' to overwrite original files, or 'Custom directory' to restore to a different path.
  • Restore MySQL — Toggle whether to restore MySQL databases (only shown if the source has MySQL backup enabled).
  • Restore PostgreSQL — Toggle whether to restore PostgreSQL databases (only shown if the source has PostgreSQL backup enabled).

Warning

In-place restore will overwrite existing files. Consider restoring to a custom directory first to verify the contents.
