Schedule Editor

Add or edit a scheduled backup.

Schedule types

  • Hourly — Run every N hours.
  • Daily — Run at a set time on selected days.
  • Weekly — Run once a week on a chosen day.
  • Monthly — Run once a month on a chosen day.
  • Custom cron — Use a raw 5-field cron expression for full control.

Fields

  • Time — The time of day to run (HH:MM).
  • Sources — Which sources to back up. Leave empty to back up all sources.
  • Destinations — Which destinations to use. Leave empty to use all destinations.
  • Retention count — Override global retention for this schedule. Leave empty for default.

Tips

  • Daily schedules let you pick specific days of the week.
  • Custom cron format: minute hour day month weekday.
