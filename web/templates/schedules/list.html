Schedules

Manage scheduled backup jobs via cron.

Table columns

  • Name — Schedule identifier.
  • Active — Whether the schedule is enabled.
  • Type — Frequency (hourly/daily/weekly/monthly/custom).
  • Time — When the backup runs.
  • Last/Next Run — Timing information.
  • Sources/Destinations — Scope of the schedule.

Tips

  • The cron daemon must be running for schedules to execute.
  • Changes are automatically synced to crontab.
