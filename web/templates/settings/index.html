Settings

Global application settings.

General

  • Log Level — Verbosity of log output.
  • Log Retention — Days to keep log files.
  • Retention Count — Default number of snapshots to keep.
  • Bandwidth Limit — Default transfer speed limit in KB/s.
  • Rsync Compression — Compress data during transfer. zstd is faster than zlib with similar ratios. Only applies to remote transfers. Note: compression adds CPU load on both sides — on fast LANs it can actually slow transfers down. Best for slow or metered links (WAN, cloud). Disable for local or gigabit LAN backups.
  • Checksum — Use file checksums instead of mtime+size to detect changes. Slower but more accurate after clock skew or filesystem migration.
  • Disk Threshold — Warn when destination disk usage exceeds this percentage.
  • Simultaneous Jobs — Max concurrent backup/restore jobs (0 = unlimited).

Notifications

  • Notify On — When to send notifications: always, failure only, or never.
  • Stale Alert Hours — Alert if any source hasn't had a successful backup in this many hours (0=disabled).
  • Digest Reports — Periodic summary of all backup jobs (daily/weekly/monthly) sent via all configured channels.
  • Email — SMTP email notifications.
  • Telegram — Send via Telegram bot. Create a bot via @BotFather.
  • Webhook — Slack, Discord, or generic webhook URL.
  • ntfy — Push notifications via ntfy.sh or self-hosted ntfy.
  • Healthchecks.io — Ping a healthchecks.io URL on success/failure.

SSH

  • Timeout — Seconds before SSH gives up.
  • Retries — Number of retry attempts.

Web Dashboard

  • Port / Host — Address and port for the web interface.
  • Password — Password used to log in to the web dashboard.
  • Login Max Attempts — Failed login attempts before lockout (default: 5).
  • Login Lockout — Seconds an IP is locked after too many failures (default: 300).

Update

Check for new versions and apply updates from the git repository. Services will restart automatically after update.

Tips

  • Per-schedule settings can override the default retention count.
