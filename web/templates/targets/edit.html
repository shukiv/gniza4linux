Source Editor

Add or edit a backup source configuration.

Fields

  • Name — Unique identifier (letters, digits, dash, underscore; max 32 chars).
  • Folders — Comma-separated paths to back up. Use Browse to pick folders interactively.
  • Include/Exclude — Glob patterns to filter files (e.g. *.conf, *.log).
  • Pre/Post hooks — Shell commands to run before/after the backup.

Source types

  • Local — Default. Back up folders on this machine.
  • SSH — Pull files from a source server via SSH/rsync.
  • S3 — Pull files from an S3-compatible bucket.
  • Google Drive — Pull files via service account.

When using a source server, specify remote paths in the Folders field instead of local paths.

MySQL

Enable MySQL to dump databases alongside files. Choose 'All databases' or select specific ones. Leave user/password empty for socket auth.

PostgreSQL

Enable PostgreSQL to dump databases alongside files. Uses pg_dump for each database and pg_dumpall for roles. Leave user/password empty for peer auth.
