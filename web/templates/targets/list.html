Sources

List and manage backup sources. A source defines what to back up: folders, file patterns, and optionally MySQL databases.

Table columns

  • Name — Unique source identifier.
  • Folders — Comma-separated paths to back up.
  • Enabled — Whether the source is active.

Tips

  • You need at least one source and one destination before you can run backups.
  • Disable a source to skip it during 'Backup All' without deleting it.
